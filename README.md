# Painting-Pad
Paint program written in Python. Breaking Bad themed. Use this paint program to create and edit your own pictures.
After finishing, user has the option to save their masterpiece.

Tools include:

  -pencil
  
  -eraser
  
  -shapes
  
  -paint
  
  -stickers
  
  -filters
